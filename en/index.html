DetoxGray Box E2E Tests and Automation Library for Mobile Apps

Getting Started

Cross Platform

Write cross-platform tests in JavaScript. Currently supports iOS, Android is nearly complete

Runs on Devices

Gain confidence to ship by testing your app on a device/simulator just like a real user

Automatically Synchronized

Stops flakiness at the core by monitoring asynchronous operations in your app

React Native Support

Built from the ground up to support React Native projects as well as pure native ones. Currently, React Native versions 0.44 and above are supported

Made For CI

Execute your E2E tests on CI platforms like Travis without grief

Test Runner Independent

Use Jest, Mocha, AVA, or any other JavaScript test runner you like

Debuggable

Modern async-await API allows breakpoints in asynchronous tests to work as expected
